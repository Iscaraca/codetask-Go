Given an integer N, the task is to find the minimum number of coins required to create all the values in the range [1, N].

Examples:

 
Input: N = 5
Output: 3
The coins {1, 2, 4} can be used to generate 
all the values in the range [1, 5].
1 = 1
2 = 2
3 = 1 + 2
4 = 4
5 = 1 + 4

Input: N = 10
Output: 4
